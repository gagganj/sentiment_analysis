Data folder contains all data needed for python notebooks to run; this needs to be in the same directory of what notebook is being used.

Neural networks require the data_processing.ipynb(do this first!)  in the same directory and for it to have been run.

Data_Exploration is used to get statistics and information of the existing data sets.

Install following packages: word cloud, collections, numpy, pandas, nltk, seaborn, matplotlib.pyplot, random, re, tensorflow, keras, sklearn,

Start a jupyter notebooks server and follow each notebook and results will be displayed in them.
